Mis Proyectos

Maxisama-notes : Proyecto final bootcamp Skillnest

Desarrollé una aplicación en Python con el framework Django bajo el patrón MTV, enfocada en la gestión organizada de tareas. El sistema permite crear, editar y administrar tareas clasificándolas por proximidad temporal (corto, mediano o largo plazo), prioridad y tipo, lo que facilita visualizar qué actividades requieren atención inmediata. La plataforma incluye un conjunto de filtros avanzados que permiten buscar tareas por categoría, estado o palabras clave, mejorando la navegación y el control del usuario. Además, incorpora un CRUD completo para gestionar referencias de textos relacionadas con cada tarea, permitiendo documentar, justificar o complementar el contenido de cada actividad. Este proyecto integra lógica backend, diseño de modelos relacionales y una interfaz limpia que facilita la planificación personal o académica.

Biblioteca Maxisamaa

Desarrollé un sistema en Django basado en el patrón MTV para gestionar y organizar modelos de libros, etiquetas y géneros, permitiendo administrar catálogos de forma ordenada y eficiente. La aplicación permite registrar, editar y clasificar libros según su género principal y múltiples etiquetas temáticas, facilitando una organización flexible adaptable tanto a tiendas como a bibliotecas. Además, integra un sistema de filtros combinados que permite buscar libros por género, etiquetas o información específica del modelo, optimizando la localización de ejemplares dentro de un inventario amplio. Este proyecto permite gestionar stock, mantener un orden lógico dentro del catálogo y ofrecer una herramienta escalable para administración literaria profesional.

Cinepedia

Cinepedia es una aplicación desarrollada en Django bajo el patrón MTV, orientada a la gestión y exploración de información cinematográfica a través de modelos de Películas, Detalles y Usuarios. El sistema funciona como una versión sencilla y didáctica de IMDb, permitiendo organizar y consultar títulos mediante un CRUD totalmente operativo. Incluye un sistema de autenticación integrado que restringe las acciones según el usuario, asegurando que solo quienes tienen permisos puedan crear, editar o eliminar contenido. Además, incorpora filtros por género, año, director u otros atributos clave, lo que facilita la búsqueda y clasificación de películas dentro del catálogo. Este proyecto demuestra la integración de modelos relacionados, control de acceso y una arquitectura ordenada para construir plataformas de consulta y administración de contenidos audiovisuales.

Recetario con autenticacion

Este proyecto es un recetario desarrollado en Django siguiendo el patrón MTV, diseñado para gestionar recetas, eventos gastronómicos y participantes asociados. Incluye un sistema de autenticación que controla el acceso al CRUD, permitiendo que solo usuarios registrados puedan crear, editar o eliminar recetas, mientras que la información pública puede consultarse libremente. El modelo principal organiza las recetas con sus ingredientes, pasos y categorías, mientras que otros modelos complementarios administran eventos y los participantes de cada uno, creando una estructura ideal para productoras culinarias o servicios de catering que necesiten una plataforma web para mostrar su oferta y, al mismo tiempo, gestionar internamente su inventario gastronómico y actividades programadas. Este proyecto destaca por su orden, correcta implementación de relaciones entre modelos y una arquitectura pensada para escalar y adaptarse a distintos tipos de negocios de comida.

Base datos de tareas organicer

Este proyecto consiste en un conjunto de sentencias SQL diseñadas para crear la estructura completa de una base de datos destinada a un organizador de tareas, funcionando como la versión “raw” de un sistema similar al desarrollado en Django. Incluye tablas para tareas, tipos, prioridades, proximidades temporales y referencias, además de todas sus relaciones mediante claves foráneas, garantizando integridad y coherencia entre los datos. La base de datos está pensada para soportar filtrado, clasificación y gestión eficiente de tareas según su importancia, urgencia y categoría, siendo una base sólida para aplicaciones futuras que necesiten construir un sistema de productividad desde cero, tanto en entornos web como de escritorio. Este proyecto demuestra un diseño limpio, normalizado y adaptable para sistemas de organización personal o profesional.
